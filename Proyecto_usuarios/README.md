# Proyecto Final Módulo 1 - Registro de Usuarios

## Descripción

Este proyecto fue desarrollado en Python y permite registrar usuarios, almacenar la información en archivos de texto y realizar validaciones básicas sobre los datos ingresados.

El programa utiliza un menú interactivo que permite registrar usuarios, mostrar los usuarios almacenados y validar registros de prueba.

---

## Funcionalidades

- Registrar usuarios.
- Validar que el nombre no esté vacío.
- Validar que la edad sea numérica.
- Evitar registros duplicados.
- Mostrar usuarios registrados.
- Validar registros desde un archivo de pruebas.
- Separar usuarios válidos e inválidos en archivos diferentes.
- Manejar errores mediante estructuras try-except.

---

## Archivos del proyecto

### registro_usuarios.py
Contiene el código principal del programa.

### usuarios.txt
Almacena los usuarios registrados correctamente.

### usuarios_pruebas.txt
Contiene registros utilizados para realizar pruebas de validación.

### usuarios_buenos.txt
Guarda los registros válidos encontrados durante la validación.

### usuarios_errores.txt
Guarda los registros que presentan errores.

---

## Tecnologías utilizadas

- Python 3
- Manejo de archivos (.txt)
- Funciones
- Condicionales
- Ciclos
- Manejo de excepciones (try-except)
